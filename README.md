# HEllO-WORlD
Text operating program fully on BASIC



   # HELLO WORLD GUIDE FOR MODDERS AND DEVELOPERS


1. If you want to make software for this FOS, you
   have to have a devs licence that you can download
   .
   if you dont have a dev's licence and you want to 
   make software, stop now. go get a licence, its 
   free.
2. Do you know how to implement the software in the
   code? If not, dont try. BASIC is easy, but not
   youre trying to implement something in another
   thing that is not compatible with it.
3. If you have the licence and you know how to
   implement your software in the code, the only
   thing that youre left to do is learn the variable
   names logic.
                        
   #        SYSTEM CODE VARIABLE NAMES LOGIC

1. @HW Paint:
   L(x)(y)x(z) = Line X in Y by Z format
  Example:
   lEL18x28$ = Line 11 in 18 by 28 format
2. @HW ProTxt:
   l(x)d(y) = line X in document Y
   Example:
   l15d2$ = line 15 in document 2
   by the way, document 1 is (T)en(l)ine(Doc)ument
   and it has 10 lines. Same for TwLdoc and ThLdoc
   (which are (Tw)enty(L)ine(Doc)ument and
   (Th)irty(L)ine(Doc)ument).
